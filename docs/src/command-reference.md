# Command help in app

Use the in-app help instead of this book for the full command list.

- Press `?`, open the `Commands` tab, and pick a command from the sidebar.
- Run `help` or `help <command>` in the minibuffer for command-specific help.
- Use [Startup scripting](./startup-scripting.md) for script files and `press ...` automation.

This keeps commands and help in one place.
